# Posture Detection App

Real-time posture monitoring application developed in Python using computer vision.

## Technologies

- Python
- OpenCV
- MediaPipe
- PyQt5

## Features

- Real-time webcam posture detection
- User calibration system
- Detection of incorrect sitting posture
- Visual feedback
- Desktop interface
